PAN Number Validation using MySQL

A SQL-based project to validate Indian PAN numbers using MySQL.
The solution cleans raw PAN data, applies regex-based format checks, and enforces business rules such as uppercase validation,
 non-sequential characters, and no adjacent repeats.

Built using CTEs, user-defined functions (UDFs), and regex, this project simulates real-world banking/fintech data quality validation and produces
 a final Valid / Invalid PAN status report.

Tech Stack: MySQL, SQL, Regex, UDFs
